# Branding Assets

Place branding images in this folder with the exact filenames below:

- `day-header-symbol.png`
- `day-header-logo.png`
- `day-footer-logo.png`
- `night-header-symbol.png`
- `night-header-logo.png`
- `night-footer-logo.png`
- `watermark-white.png`

Notes:

- Use transparent PNG for all files.
- `watermark-white.png` should be the combined symbol + white Chrono Lab logo image.
- If a file is missing, the app falls back to current admin-saved settings or text fallback.
